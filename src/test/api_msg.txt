1. api.function
    - 可能有注释，那么就直接获取分析注释即可
    - 如果没有注释，我们也可以进行自主分析，不过没有注释给出的那么全
    function名、参数
/**
* @desc xx
* @name xxx
* @param xxx
* @return
*/

2. $.ajax
    同上，有就找注释，没有自行生成
    其内容信息在$.ajax(...)里面，url、data、type、dataType
